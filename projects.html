HRM Genotyping (Python)

I started this project as a way to quickly be able to analyze the melting curves generated by qPCR for genotyping mice (read about the process of generating and analyzing HRM curves here). This program was a simple way to transform all of the raw data, in a tab-delimited text file, to individual excel sheets by gene marker as well as generate folders and graphs of the analyzed data. The parsing and generation of files is all done in Python and the analysis portion of the code is sourced from an open-source code released by Liu Yi on Github.
